# 20260215 - レイアウトバリエーション + リンク集ページ作成

## 概要

既存のサイバー風ポートフォリオ(v1)に加え、同じテーマ・コンテンツでレイアウトが異なる2つのバリエーションを作成し、比較用のリンク集ページを設置した。

## 作業内容

### 1. 既存ファイルを `v1/` に移動

- `index.html`, `style.css`, `script.js` を `v1/` ディレクトリにコピー
- 縦スクロール・中央揃え・セクション分割のスタンダードレイアウト

### 2. `v2/` サイドバーナビレイアウト作成

- 左に固定サイドバー（ロゴ・ナビリンク・SNSリンク）
- 右にコンテンツエリア（Hero / About / Gallery）
- ギャラリーはMasonry風2カラム（`columns: 2`）
- スクロール位置に応じてナビリンクのアクティブ状態を自動更新
- モバイル(768px以下)ではサイドバーを非表示にし、上部ヘッダー + ハンバーガーメニューに切り替え

### 3. `v3/` フルスクリーンスクロールレイアウト作成

- 各セクション(Hero / About / Gallery)が `100vh` のフルスクリーン表示
- `scroll-snap-type: y mandatory` でページめくり感を実現
- 右側にドットナビゲーション（スクロール位置に連動）
- ギャラリーは横スクロールカルーセル風（`scroll-snap-type: x mandatory`）
- Heroセクションに「Scroll」ヒントアニメーション付き
- モバイル(768px以下)ではスナップスクロールを無効化し通常スクロールにフォールバック

### 4. ルート `index.html` をリンク集ページに差し替え

- サイバー風テーマ統一のデザイン
- 3つのバリエーションのプレビューカード（CSSで描画したミニチュアモックアップ付き）
- 各カードにバージョンバッジ・レイアウト名・説明・Viewリンクボタン
- スタイルはインラインCSS（外部ファイル不要）
- ルートの `style.css`, `script.js` を削除

## 最終ディレクトリ構成

```
/
├── index.html          ← リンク集トップページ
├── v1/
│   ├── index.html      ← スタンダード（現行デザイン）
│   ├── style.css
│   └── script.js
├── v2/
│   ├── index.html      ← サイドバーナビ + Masonry 2カラム
│   ├── style.css
│   └── script.js
└── v3/
    ├── index.html      ← フルスクリーンスナップスクロール + カルーセル
    ├── style.css
    └── script.js
```

## 共通仕様（全バリエーション）

- カラーテーマ: サイバー風（ダーク背景 + シアン/マゼンタ/パープルのアクセント）
- コンテンツ: Hero / About / Gallery の3セクション構成
- ギャラリー: 8枠の「Coming Soon」プレースホルダー
- Lightbox機能（`data-src` 属性で画像指定可能）
- レスポンシブ対応（768px以下でモバイルレイアウトに切り替え）
- SNSリンク: X(@BuBu_AIIlust)
